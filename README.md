# M&E Device Status Automation

This project automates the monitoring of M&E (Mechanical & Electrical) devices using Python. 

## What this code does:
- **Filters Data:** Automatically detects devices with "Down" status.
- **Generates Reports:** Saves faulty device reports to an Excel file.
- **Visualization:** Creates a bar chart to visualize the operational status of all devices.

## Requirements:
- Python
- Pandas
- Matplotlib

## How to use:
Simply run the script in a Jupyter Notebook or Google Colab environment to generate the latest status report.
